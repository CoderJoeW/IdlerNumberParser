Introduction

IdlerNumberParser(IDN) is a simple lightwieght frame work for converting numbers for display. Our simple parser can take a large number such as 65656554 and turn it into 65.7M for display while still retaining the value and all decimal places. This is the first release of the parser but we will continue improving it making it faster and provide higher values! Currently IDN supports numbers up to 999 Decillion

Usage

Download the file from our github repo and put it in a folder named "js" on your server.
Once on your server add this to the HEAD of your document 

    
      <script src="js/IdlerNumberParser.js"></script>
Once done you may use the function by calling INP(number,round);

Documentation

```javascript INP(number,round); ``` INP Supports rounding and non rounding. You can make a call to INP only filling in the number parameter which will return the raw number back with its prefix for example ```javascript INP(5003); ``` will return `5.003K` you can specify how many decimal places should be rounded by using filling in the round parameter for example ```javascript var a = INP(5123,0); var b = INP(5123,1); var c = INP(5123,2); var d = INP(5123,3); console.log(a); console.log(b); console.log(c); console.log(d); ``` the output to the console would be ` 5K 5.1K 5.12K 5.123K `
